# MM — 메인 메뉴 (Main Menu)

## 개요

프로그램 진입점. 역할 선택 후 각 역할별 메뉴로 진입하며, 역할 메뉴 상단에 전체 시료 요약 정보를 표시한다.

---

## 화면 구성

### 역할 선택 화면 (진입점)

```
========================================
  반도체 시료 생산주문관리 시스템
========================================
 1. 주문 담당자
 2. 생산 담당자
 0. 종료
========================================
선택: _
```

### 주문 담당자 메뉴

```
========================================
  [주문 담당자]  시료: N종  총 재고: N개
========================================
 1. 주문 접수
 2. 접수 주문 목록
 3. 출고 처리
 4. 모니터링
 0. 역할 선택으로 돌아가기
========================================
선택: _
```

### 생산 담당자 메뉴

```
========================================
  [생산 담당자]  시료: N종  총 재고: N개
========================================
 1. 시료 관리
 2. 주문 승인 · 거절
 3. 생산 라인
 4. 모니터링
 0. 역할 선택으로 돌아가기
========================================
선택: _
```

---

## 역할별 기능 매핑

### 주문 담당자

| 입력 | 동작 |
|------|------|
| 1 | 주문 접수 → [OR-01](OR-order.md) |
| 2 | 접수 주문 목록 → [OR-02](OR-order.md) |
| 3 | 출고 처리 서브 메뉴 → [RL](RL-release.md) |
| 4 | 모니터링 서브 메뉴 → [MO](MO-monitoring.md) |
| 0 | 역할 선택으로 복귀 |
| 그 외 | "올바른 메뉴를 선택해주세요" 출력 후 재표시 |

### 생산 담당자

| 입력 | 동작 |
|------|------|
| 1 | 시료 관리 서브 메뉴 → [SM](SM-sample-management.md) |
| 2 | 주문 승인·거절 서브 메뉴 → [OR-03, OR-04](OR-order.md) |
| 3 | 생산 라인 서브 메뉴 → [PL](PL-production-line.md) |
| 4 | 모니터링 서브 메뉴 → [MO](MO-monitoring.md) |
| 0 | 역할 선택으로 복귀 |
| 그 외 | "올바른 메뉴를 선택해주세요" 출력 후 재표시 |

## 처리 흐름

```
프로그램 시작
    │
    └─ 역할 선택 루프
            │
            ├─ 0 → 프로그램 종료
            ├─ 1 → 주문 담당자 메뉴 루프
            │           │
            │           ├─ 요약 정보 조회 및 메뉴 출력
            │           ├─ 1~4 → 해당 기능 실행 → 복귀
            │           ├─ 0 → 역할 선택으로 복귀
            │           └─ 그 외 → 오류 메시지 → 재출력
            ├─ 2 → 생산 담당자 메뉴 루프
            │           │
            │           ├─ 요약 정보 조회 및 메뉴 출력
            │           ├─ 1~4 → 해당 기능 실행 → 복귀
            │           ├─ 0 → 역할 선택으로 복귀
            │           └─ 그 외 → 오류 메시지 → 재출력
            └─ 그 외 → 오류 메시지 → 역할 선택 재출력
```
