# MM — 메인 메뉴 (Main Menu)

## 개요

프로그램 진입점. 전체 시료 요약 정보를 상단에 표시하고, 기능별 서브 메뉴로 진입하는 네비게이션을 제공한다.

---

## 화면 구성

```
========================================
  반도체 시료 생산주문관리 시스템
  [요약] 시료: N종  총 재고: N개
========================================
 1. 시료 관리
 2. 주문 (접수 / 승인 / 거절)
 3. 모니터링
 4. 출고 처리
 5. 생산 라인
 0. 종료
========================================
선택: _
```

---

## 요약 정보 표시

메뉴 출력 전 매번 최신 데이터를 조회해 갱신한다.

| 항목 | 계산 방법 |
|------|-----------|
| 시료 수 | `SampleRepository::findAll().size()` |
| 총 재고 | 전체 시료의 `stock` 합산 |

---

## 메뉴 선택 처리

| 입력 | 동작 |
|------|------|
| 1 | 시료 관리 서브 메뉴 진입 → [SM](SM-sample-management.md) |
| 2 | 주문 서브 메뉴 진입 → [OR](OR-order.md) |
| 3 | 모니터링 서브 메뉴 진입 → [MO](MO-monitoring.md) |
| 4 | 출고 처리 서브 메뉴 진입 → [RL](RL-release.md) |
| 5 | 생산 라인 서브 메뉴 진입 → [PL](PL-production-line.md) |
| 0 | 프로그램 종료 |
| 그 외 | "올바른 메뉴를 선택해주세요" 출력 후 재표시 |

---

## 처리 흐름

```
프로그램 시작
    │
    └─ 루프
            │
            ├─ 요약 정보 조회 및 메뉴 출력
            ├─ 입력 대기
            │
            ├─ 0 입력 → 루프 종료 → 프로그램 종료
            ├─ 1~5 입력 → 해당 서브 메뉴 실행 → 완료 후 메인 메뉴로 복귀
            └─ 그 외 → 오류 메시지 → 메뉴 재출력
```

## 관련 Repository
